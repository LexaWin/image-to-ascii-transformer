# Roadmap

## Stage: Base 

1. browser -> image // ~100 lines code
2. image -> canvas  // 1 line code
3. canvas -> pixel  // 1 line code
4. pixel -> ascii code // 
    - pixel RGB (255, 255, 255) -> grayscale (0..255)
    - grayscale value -> alphabet symbol
